Documentation & Methodology

Data Sources

  • Global Forest Watch (GFW): Tree cover loss data (v1.11) using Hansen et al. (2013) methodology.
  • IPIS (International Peace Information Service): Mining concession boundaries and status data for DRC.
  • OWR Authority: Official boundary and management zone GeoJSON (2024 update).

Processing Workflow

The analysis is conducted using a multi-stage geospatial pipeline:

  1. Spatial Join: Concession polygons are intersected with the OWR boundary and 50km buffer.
  2. Aggregation: Annual forest loss is calculated for each management zone using Python (Pandas/GeoPandas).
  3. Validation: Results are cross-referenced with Landsat 8/9 spectral imagery for hotspot verification.
